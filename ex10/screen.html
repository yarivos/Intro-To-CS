Screen documentation¶

In this module you will find the methods needed for interacting with the UI

class screen.Screen¶
__init__()¶

This inits our graphics class.

draw_asteroid(asteroid, x, y)¶

Draw the given asteroid on the specified (x,y) coordinates

Parameters:
  • asteroid (Asteroid) – This is your asteroid object (remember to register it before)
  • x (int) – This is the X coordinate of the asteroid
  • y (int) – This is the Y coordinate of the asteroid
draw_ship(x, y, heading)¶

Draw the ship at the given coordinates with the given heading

Parameters:
  • x (int) – This is the X coordinate of the ship
  • y (int) – This is the Y coordinate of the ship
  • heading (float) – This is the heading of the ship (in degrees)
draw_torpedo(torpedo, x, y, heading)¶

Draw the given torpedo on the specified (x,y) coordinates with the given heading

Parameters:
  • asteroid (Torpedo) – This is your torpedo object (remember to register it before)
  • x (int) – This is the X coordinate of the torpedo
  • y (int) – This is the Y coordinate of the torpedo
  • heading (float) – This is the heading of the torpedo
end_game()¶

This ends the current game.

is_left_pressed()¶
Returns:True if the left key was pressed, else False
is_right_pressed()¶
Returns:True if the right key was pressed, else False
is_space_pressed()¶
Returns:True if the fire key was pressed, else False
is_special_pressed()¶
Returns:True if the fire key was pressed, else False
is_up_pressed()¶
Returns:True if the up key was pressed, else False
is_teleport_pressed()¶
Returns:True if the 't' key was pressed, else False
ontimer(func, milli)¶

This method is used to create a repeating action in your game.

Warning

You don’t need to call this method, it was already called for you at the end of the main game loop.

Parameters:
  • func (function) – The function to repeat after milli milliseconds have passed
  • milli (int) – The amount of milliseconds to wait before starting the given function
register_asteroid(asteroid, size)¶

This is called to register a new asteroid in our system

Parameters:
  • asteroid (Asteroid) – This is your asteroid object
  • size (int) – The size of the asteroid (this should be in [1,2,3])
register_torpedo(torpedo)¶

This is called to register a new torpedo in our system

Parameters:asteroid (Torpedo) – This is your torpedo object
remove_life()¶

Remove one icon of life (starts with 3 lives)

set_score(val)¶

Sets the current game score

Parameters:val (int) – The game score
should_end()¶
Returns:True if the game should end or not (if “q” was pressed or not)
show_message(title, msg)¶

This is a method used to show messages in the game.

Parameters:
  • title (str) – The title of the message box.
  • msg (str) – The message to show in the message box.
start_screen()¶

This is called to start our game (grphaics-wise).

Warning

This method should not be called by you

unregister_asteroid(asteroid)¶

This is called to un-register an existing asteroid in our system

Parameters:asteroid (Asteroid) – This is your asteroid object
unregister_torpedo(torpedo)¶

This is called to un-register an existing torpedo in our system

Parameters:asteroid (Torpedo) – This is your torpedo object
update()¶

This is called to update our game (grphaics-wise).

Warning

This method should not be called by you
